# Configure npm

Modify configuration of npm to add proxy to download javascript npm libraries

Open a console and type

```bash
npm config set proxy http://user:pass@proxy2.inti.com:3128
npm config set https-proxy http://user:pass@proxy2.inti.com:3128
```

To check the the configuration added. Access to **_~/user/.npmrc_** and verify that the proxies have been set properly.
